# Printables

The files in this directory are what you need to print (or have printed) to complete the kit.

## Choosing The Right Files
A few options are provided, so you need to be selective in which files you print:
- There are [Branded](/Printables/Branded) and [Unbranded](/Printables/Unbranded) options are provided; so you can decide if you want the `GW` logo or not. You only need to print one set of files
